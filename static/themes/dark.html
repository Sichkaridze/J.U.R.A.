About J.U.R.A.

J.U.R.A. is a modern, interactive task management tool designed for IT teams. With an emphasis on playfulness and efficiency, it helps you organize tasks, manage teams, and track progress in a fun and engaging way.

  • Back-end: Django (Python)
  • Front-end: Bootstrap
  • Database: SQLite / PostgreSQL
  • Deployment: Docker / AWS (future)
